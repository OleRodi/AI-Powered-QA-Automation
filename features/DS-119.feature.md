Feature: DS-119 — Dashboard displaying the right components
  As an admin user, I want to see the correct dashboard so that I can navigate to key areas of Didaxis Studio.

  # Happy paths

  Scenario: Navigate to the Dashboard
    Given I am logged in as admin
    When I navigate to the Dashboard page
    Then I see the Dashboard with the right blocks: Programs, Calendar, Validation, AI Assist

  Scenario: Successfully navigate to Program Page
    Given I am on Dashboard
    When I click on Programs card
    Then I navigate to the Programs page

  Scenario: Successfully navigate to Calendar Page
    Given I am on Dashboard
    When I click on Calendar card
    Then I navigate to the Calendar page

  Scenario: Successfully navigate to Validation Page
    Given I am on Dashboard
    When I click on Validation card
    Then I navigate to the Validation page

  Scenario: Successfully navigate to AI Assist Page
    Given I am on Dashboard
    When I click on AI Assist card
    Then I navigate to the AI Assist page

  # Negative

  Scenario: Dashboard cards do not navigate when only viewing static content
    Given I am on Dashboard
    When I have not clicked a dashboard card
    Then I remain on the Dashboard page

  # Edge cases

  Scenario: Each dashboard block shows descriptive subtitle text
    Given I am on Dashboard
    Then the Programs block shows "Manage academic programs"
    And the Calendar block shows "Schedule & drag-drop"
    And the Validation block shows "Check for conflicts"
    And the AI Assist block shows "AI-powered editing"
